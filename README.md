
# 🚀 BioMark — AI-Powered Multimodal Biometric Authentication System

Secure identity verification using **Face Recognition** and **Voice Authentication** for fast, reliable, and spoof-resistant authentication.

---

## 📌 Overview

**BioMark** is an AI-powered multimodal biometric authentication system that verifies a user's identity by combining **facial recognition** and **voice authentication**. Unlike traditional single-biometric systems, BioMark uses dual verification to significantly improve security, accuracy, and reliability.

The system is designed to reduce spoofing attacks while providing a seamless and user-friendly authentication experience suitable for educational institutions, organizations, and secure access control systems.
